Why Athvexa

Built for the pitch.
Designed for the coach.

Everything a football strength coach needs to track, analyse, and improve player performance at every level.

📊
Real-time dashboards

Live view of every player's daily score, wellness status, and task completion — before you even step on the pitch.

📴
Offline-first

Players can log data without internet. Everything syncs automatically when they reconnect. No excuses, no lost entries.

🌍
Multilingual

English by default. Switch to Arabic or Persian in settings. Gregorian, Shamsi, and Hijri calendars all supported.

🔔
Smart reminders

Automated notifications for training sessions, habit check-ins, and wellness assessments — configured by the coach.

🔗
Invite via link

Add players instantly with a shareable invite link. No admin friction, no account setup required on the player's side.

🛡️
Secure & private

Self-hosted on your own server. JWT authentication, HTTPS enforced, full data backup and export control in your hands.

Access roles

Three roles.
One platform.

From head coach to player — each person sees exactly what they need, nothing more.

Head Coach

Full Control

Complete access to all players, all data, all settings. Assign training, set goals, manage the entire team from one screen.

✓ Manage all players
✓ Add assistants
✓ All analytics
✓ Full settings access
Assistant Coach

Delegated Access

Head coaches can delegate specific players or tasks to assistants — even granting partial coach-level access where needed.

✓ Assigned players
✓ Assign training
✓ Send coach messages
✓ Configurable scope
Player

Own Dashboard

Players see their own data, check in daily, complete tasks, track habits, and receive messages from the coaching staff.

✓ Personal dashboard
✓ Daily check-in
✓ Habit tracking
✓ Coach messages

Build plan

From zero to launch
in three phases.

Phase 1 — Foundation

Core app

  • Auth (email + password, remember me)
  • athvexa.com landing page
  • app.athvexa.com player dashboard
  • Today Dashboard + Daily Routine
  • Coach player management
  • Invite via link
  • PWA + offline sync
Phase 2 — Intelligence

Full feature set

  • Weekly Planner + Habits module
  • Goals + Wellness Check-in
  • Notification system
  • Assistant coach roles
  • Performance analytics
  • Multi-language (AR/FA)
  • Calendar system choice
Phase 3 — Scale

Growth ready

  • Multi-tenant (sell to other coaches)
  • Subscription billing
  • Data export / import
  • Light theme
  • Advanced injury tracking
  • Video clip attachments
  • Custom domain per tenant
